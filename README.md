Log-Work
============

Log Your Daily Work is an productivity web application.

Technology
============
Programming  : Java, JDBC
Framework    : Spring Framework
Database     : H2 Database Engine
View         : Thymeleaf, HTML5, CSS3
Build        : Gradle, Maven
IDE          : Spring Tool Suite
